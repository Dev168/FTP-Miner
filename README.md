FTP-Miner
=========

An FTP search engine scraper, currently the following search engines are supported:

    -   Napalm FTP indexer (http://www.search-ftps.com/)
    -   Mamont (http://www.mmnt.ru/)
    -   FileWatcher (http://www.filewatcher.com/)
    -   FileMare (http://filemare.com/)

If you are aware of another big ftp search engine which I should take a look at,
feel free to contact me.
